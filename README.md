# Fractal Renderer

A C++ fractal renderer created to explore Mandelbrot, Julia and experimental fractals.

## 7 unique types of fractals
 ***Mandebrot types***
    - *Classic*
    - *Raised to power 3*
    - *Tricorn or Mandelbar*
    - *Cosine function*
    - *Sine function*

 ***Julia set types***
    - *Classic*
    - *Flower*
    - *Cone*
    - *Sprial*
    - *Snake*
    - *Silver ratio*
